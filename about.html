What is IP-HUB?

IP-HUB is a real-time cybersecurity threat intelligence dashboard that visualizes global cyber threats on an interactive 3D globe. It allows users to see live blacklisted IPs pulsing on the map, search any IP address to check its threat level and confidence score, and pinpoint the exact city location of the given IPs. It also shows a heat map of the countries and thier respective network traffic (mock data) .


Frontend

IP-HUB Frontend is built using Three.js and the Three-Globe library to create an interactive 3D visualization of global cyber threats. Users can hover over any marker to view information

The globe is constructed by loading a custom.geo.json file containing GeoJSON-formatted geographic data that defines the boundaries of countries and territories around the world.


Backend

IP-HUB Backend is built using FastAPI, a high-performance Python web framework. It provides endpoints that fetch real-time data from AbuseIPDB, including IP reputation, confidence scores, total reports, and attack categories.

For city-level geolocation, the backend integrates with ip-api.com to retrieve precise latitude and longitude coordinates for any IP address. The backend also includes private IP detection, input validation, error handling, and rate limit management.
